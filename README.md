# PlyViewer

PlyViewer is a web-based 3D model viewer for PLY files. It is built using Three.js.

## Custumization
index.html add button
menu.ts add event listener for button

## CLI Commands

Installation

[node.js](https://nodejs.org/) and npm are required.

```bash
npm i
```

Run dev mode

```bash
npm run dev
```

Build

```bash
npm run build
```

Run build

```bash
npm run preview
```
